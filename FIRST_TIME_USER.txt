🅿️ PARKING MANAGEMENT SYSTEM - FIRST TIME USER GUIDE
=====================================================

Welcome! This is a complete, professional parking management system 
with a beautiful web dashboard and real-time features.

⏱️ SETUP TIME: 5-10 minutes
═════════════════════════════════════════════════════════════════════

STEP 1️⃣: DOWNLOAD & EXTRACT
────────────────────────────
You're already here! All files are ready to use.

STEP 2️⃣: INSTALL PYTHON
────────────────────────
Windows: Download from python.org (Python 3.8+)
Mac: brew install python3
Linux: apt-get install python3 python3-pip

Verify: Open terminal/cmd and type:
  python --version    (Windows)
  python3 --version   (Mac/Linux)

STEP 3️⃣: START THE SYSTEM
─────────────────────────

EASY METHOD (Recommended):
  Windows: Double-click START_SYSTEM.bat
  Mac/Linux: Run in terminal: bash start_system.sh
  
MANUAL METHOD:
  Open terminal/cmd in this folder, type:
  python run_system.py

ADVANCED METHOD:
  python app_ui.py

STEP 4️⃣: WAIT FOR SUCCESS
──────────────────────────
You'll see:
  ✓ Python version check
  ✓ Installing dependencies...
  ✓ Creating directories...
  ✓ Database initialized...
  ✓ Starting Flask...
  
Browser will auto-open at http://localhost:5000

STEP 5️⃣: EXPLORE THE DASHBOARD
────────────────────────────────
You're now in! You should see:
  
  📊 DASHBOARD (http://localhost:5000)
     • Real-time statistics
     • Active vehicles
     • Recent exits
     • Search by plate
     
  🎛️ CONTROL PANEL (http://localhost:5000/control)
     • System status
     • Database management
     • Data export
     • System logs
     
  📈 REPORTS (http://localhost:5000/reports)
     • Daily summary
     • Revenue chart
     • Transaction history
     • Advanced search

⚡ TROUBLESHOOTING:
────────────────────
Issue: "Port already in use"
Fix: Change port in app_ui.py (last line, change 5000 to 5001)

Issue: "Module not found"
Fix: pip install -r requirements.txt

Issue: "Database locked"
Fix: Delete parking.db and restart

Issue: "Page won't load"
Fix: Check terminal for errors, ensure Flask says "Running on"
